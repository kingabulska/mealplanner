Description

Let's start with something simple. Write a program that can save a meal and its properties. Prompt the user about the meal category (breakfast, lunch, or dinner), its name, and the necessary ingredients. The program should save that information and be able to print the saved meal with all its properties in the correct format.

Objectives

In this stage, your program should:

  1. Ask the user about the meal category with the following message: Which meal do you want to add (breakfast, lunch, dinner)?.

  2. Save the category and ask about the name of the meal with the message Meal's name:.

  3. Save the meal's name and then ask about the necessary ingredients with the message Ingredients:. The ingredients should be listed in one line separated by commas.

  4. Save the ingredients and print all the information about the meal in the following format:

Category: category
Name: meal's name
Ingredients:
ingredient 1
ingredient 2
etc.

5. Print the message that the meal was successfully saved: Meal added!.

Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1

Which meal do you want to add (breakfast, lunch, dinner)?
> lunch
Meal's name:
> salad
Ingredients:
> lettuce, tomato,onion, cheese, olives
Category: lunch
Name: salad
Ingredients:
lettuce
tomato
onion
cheese
olives
Meal added!
