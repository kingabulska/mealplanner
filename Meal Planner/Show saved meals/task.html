Description

Let's make it easier for the user to navigate the program and find exactly what they need. In this stage, we will work on the command show. The program should prompt the user for the meal category with the message Which category do you want to print (breakfast, lunch, dinner)?. Then, it should search the database and print only the meals from the chosen category. If the requested category does not exist, the program should show an appropriate message.

Objectives

1. Prompt the user for the meal category.

2. Select the names of meals of the chosen category.

3. Print Category: <category> followed by the list of meals, each on a new line.

4. If the input category does not exist, print Wrong meal category. Choose from: breakfast, lunch, dinner.

5. If the chosen category has no meals, print No meal found.

Example

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.
