Description

A good planner plans for the week ahead! Let's help our users plan their meals for the entire week. For this, we will add a command plan to our menu. The program should print the first day of week – Monday – and then print the list of all breakfast meals stored in the database in alphabetical order. Then, the program should prompt the user to pick a meal option with the message Choose breakfast for Monday from the list above:.

After the user provided a name, the program should verify it. If the database does not contain the meal with such a name, print This meal doesn’t exist. Choose a meal from the list above.. If the user input is correct, move on to the next category – lunch – and then to dinner. After the user has picked the meal options for all three categories, print Yeah! We planned the meals for Monday..

Repeat these steps for the rest of the days all the way through Sunday. In the end, print the whole weekly plan.

Objectives

1. Insert an empty line after one day's meals have been planed.

2. The format should be as follows:

Monday
Breakfast: <meal's name>
Lunch: <meal's name>
Dinner: <meal's name>

Tuesday etc.

Example

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Note: the example below assumes that the database already has quite a few meals stored.
